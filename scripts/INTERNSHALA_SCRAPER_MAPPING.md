# Internshala Scraper → Schema Field Mapping

## Fields Scraped from HTML

### From Listing Page

- `name` → title from `a.job-title-href`
- `company` → from `p.company-name`
- `location` → from `div.row-1-item.locations`
- `is_remote` → inferred from location text (work from home/remote/wfh)
- `apply_link` → constructed from href
- `stipend` → parsed from `span.stipend` (e.g. "₹ 6,000 /month")
    - `type`: "paid" | "unpaid" | "performance-based"
    - `amount`: numeric value (lower bound if range)
    - `currency`: "INR" | "USD" | "GBP" | "EUR"
    - `period`: "monthly" | "weekly" | "lump-sum"
- `duration_string` → from 3rd `div.row-1-item > span` (e.g. "3 Months")

### From Detail Page (fetched for each internship)

- `city` → from `div#location_names > span > a`
- `duration_string` → from `div.other_detail_item` with "Duration" heading (overrides listing)
- `deadline_text` → from `div.other_detail_item.apply_by` (e.g. "3 Jun' 26")
- `skills` → from `h3.skills_heading` + `div.round_tabs_container > span.round_tabs`
- `perks` → from `h3.perks_heading` + `div.round_tabs_container > span.round_tabs`
- `openings` → from "Number of openings" heading + `div.text-container`
- `responsibilities` → from `h2.about_heading` + `div.text-container` (split into array)
- `about_company` → from company heading + `div.about_company_text_container`
- `company_website` → from `div.website_link > a[href]`

## Fields from AI Enrichment (Cohere)

- `degree` → array of degree requirements
- `field` → array of study fields
- `summary` → 50-70 word summary (generated from responsibilities)
- `country` → defaults to "india"
- `city` → fallback if not in HTML

## Fields Set by Scraper

- `date_published` → current timestamp (ISO format)
- `deadline_date` → parsed from `deadline_text` (ISO format)
- `source` → "web_scraping"
- `state` → null (not available in Internshala HTML)

## Fields Handled by Pipeline

- `fingerprint` → generated by deduplicator
- `linkVerification` → populated by linkVerifier
- `moderation` → populated by scorer/validator
- `duration` → parsed from `duration_string` by normalizer
- `experienceRequired` → defaults to {unit: "months"}
- `isActive` → defaults to true
- `tags` → null

## Schema Compliance

All required fields are populated:

- ✅ name, company, applyLink
- ✅ datePublished, deadlineDate (parsed)
- ✅ country, city, isRemote
- ✅ stipend (with correct period: "monthly" not "month")
- ✅ duration (as duration_string, parsed by normalizer)
- ✅ skills (array from detail page)
- ✅ degree, field (from AI)
- ✅ summary (from AI)
- ✅ responsibilities (array, not string)
- ✅ perks (array)
- ✅ openings (number)
- ✅ source

## Known Limitations

- `state` is not available in Internshala HTML
- `experienceRequired` is not extracted (could be in "Who can apply" section)
- `tags` are not generated
- Detail page fetch adds ~0.5s per internship (rate limiting)
